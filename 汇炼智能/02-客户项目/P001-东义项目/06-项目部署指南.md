## 🔬 第二阶段:预研验证(当前)

### 2.1 预研安排

#### 现场调研计划
**时间**: 2025年11月3日（2-3天）
**地点**: 东义集团现场

**参与人员**:
- 极峰科技: 算法工程师 × 1 + 智能体产品专家 × 1
- 汇炼智能: 我方陪同协调

**调研对象**:
- 合成氨分厂
- 甲醇分厂
- 干熄焦分厂
- AI配煤相关部门

#### 调研目标
1. **数据收集**: 各分厂生产数据、工艺参数、历史记录
2. **业务访谈**: 了解业务场景、痛点、期望目标
3. **技术验证**: 确认各智能体技术可行性
4. **方案细化**: 明确开发难度、工作量、技术路线

#### 调研准备清单
**极峰准备**:
- [ ] 智能体技术调研问卷
- [ ] 数据采集接口调研清单
- [ ] 技术环境检查清单
- [ ] 工艺参数调研模板

**东义准备**:
- [ ] 各分厂DCS系统资料
- [ ] 历史生产数据样例
- [ ] 现有工艺流程文档
- [ ] IT基础设施情况说明
- [ ] 各分厂对接人安排

**我方准备**:
- [ ] 调研行程安排
- [ ] 调研问题清单
- [ ] 与东义确认接待安排
- [ ] 协调极峰团队出差

### 2.2 预研产出

极峰将在11月中旬提交:

**1. 预研报告**:
- 各智能体技术可行性分析
- 数据基础评估
- 技术难点识别
- 风险评估

**2. 技术方案**:
- 详细的开发方案和架构设计
- 各智能体功能模块设计
- 系统集成方案
- 技术路线图

**3. 工作量评估**:
- 准确的开发周期估算
- 资源需求分析
- 成本核算

**4. 合作方案**:
- 正式报价
- 实施计划
- 验收标准
- 质保条款

**存放位置**: `03-项目执行/文档/预研报告/`

### 2.3 预研后决策

**待决策事项**:
1. **甲醇智能体是否进入第一期？**
   - 依据: 预研报告中的技术难度、开发周期、价值评估
   - 决策时间: 11月中旬（预研报告后）

2. **第一期总价确定**:
   - 45万（不含甲醇）还是 85万（含甲醇）
   - 依据: 预研报告的工作量评估

3. **与极峰的分成比例**:
   - 等待极峰正式报价后协商
   - 明确各自工作量和价值贡献

4. **合同签署方式**:
   - 采用三方协议还是两份合同
   - 与极峰、东义法务沟通确定

5. **付款方式和周期**:
   - 根据开发周期和里程碑设计
   - 平衡现金流和客户压力

**详细预研计划**: 见 [02-商务阶段/框架协议谈判.md](02-商务阶段/框架协议谈判.md#预研阶段)

## 💰 第三阶段:商务谈判(预研后)

### 3.1 报价策略制定

**成本核算**(待预研报告后确认):
- 极峰技术开发成本: 根据预研报告确定
- 我方成本: 商务、项目管理、售后
- 框架协议总价: 预计45-85万（看甲醇是否进入一期）

### 3.2 分期签约策略(推荐)

**第一期合同**（先签）:
- 合成氨智能体（约30万）
- 甲醇合成氨分厂数据聚合（约15万）
- AI配煤PoC（0万，战略项目）
- 可能包含：甲醇智能体（约40万）- 看预研结果

**第二期合同**（一期完成后签）:
- 干熄焦智能体
- 其他根据一期效果和新需求补充

### 3.3 付款方式(初步方案)

**方式A: 里程碑付款（推荐）**
- 签约时: 30%
- 中期里程碑: 30%（如数据接入完成、核心功能上线等）
- 验收时: 30%
- 质保期满: 10%

**方式B: 简化分期**
- 签约时: 40%
- 验收时: 50%
- 质保期满: 10%

**具体方式待预研报告后确定**

## ✅ 第五阶段:验收与交付(执行完成后)

### 5.1 项目验收

**验收标准**:
- 功能满足需求文档
- 性能达到约定指标
- 通过东义业务人员验收
- 交付完整文档与培训

**验收流程**:
1. 极峰演示系统功能
2. 东义用户操作验证
3. 功能清单逐项确认
4. 文档交付确认
5. 签署验收报告

## 🛠️ 第六阶段:售后服务(验收后1年)

### 6.1 售后服务范围

**免费服务（1年）**:
- 4小时响应（工作时间）
- 远程协助为主
- 免费Bug修复
- 定期健康检查（每季度1次）

**问题分级**:
| 级别 | 定义 | 响应时间 | 解决时间 |
|-----|------|---------|---------|
| P0-紧急 | 系统无法使用 | 1小时 | 4小时 |
| P1-高 | 核心功能故障 | 2小时 | 8小时 |
| P2-中 | 次要功能故障 | 4小时 | 2天 |
| P3-低 | 优化建议 | 8小时 | 1周 |

### 6.2 定期回访

- 月度回访（电话/微信）
- 季度健康检查（现场/远程）
- 年度总结（现场）
