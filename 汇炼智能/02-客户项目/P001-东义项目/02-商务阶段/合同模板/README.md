## 🔗 其他相关合同模板

### 极峰原版联合研发协议
**存放路径**: `../05-项目文档-极峰提供/合同模板/联合研发协议-极峰原版-需改造.doc`

**重要说明**: ⚠️ 该模板原本是极峰直接与甲方（如东义）签约的模式，需要改造后使用

**改造方向**:
- **方案1**: 改为汇炼（甲方）-极峰（乙方）的分包合同
  - 将原合同中"极峰-东义"的法律关系改为"汇炼-极峰"
  - 添加背靠背（back-to-back）条款，确保风险传导
- **方案2**: 作为技术规格附件纳入分包合同
  - 分包合同引用该文档中的技术条款和交付标准
- **方案3**: 仅参考技术条款，汇炼自行起草分包合同

**对应合同架构**: 合同2B - 汇炼-极峰 煤化工领域独家战略合作协议

## ✅ 使用建议

### 1. 使用前必做事项
- [ ] 所有模板使用前需经过法务审查
- [ ] 确认汇炼作为总包方的定位在合同中得到体现
- [ ] 特别注意"联合研发协议"需要改造甲乙方关系

### 2. 背靠背条款原则
确保汇炼-东义总包合同与汇炼-极峰分包合同的条款一致：
- 技术指标要求一致
- 交付标准一致
- 验收条件一致
- 违约责任对等

### 3. 知识产权注意事项
- 明确项目成果的知识产权归属
- 争取汇炼在煤化工领域的主导权
- 保护极峰的背景技术

### 4. 合同保存规则
- **模板文件**: 保存在本文件夹
- **正式签署的合同**: 存档在 `02-商务阶段/文档/` 或创建独立的合同归档文件夹
- **合同修改版本**: 保存在 `02-商务阶段/_历史版本/`
