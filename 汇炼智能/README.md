## 🚀 快速开始

### 1. 日常工作入口

**每天开始工作**:
1. 打开 `00-公司工作台.md` - 查看整体运营状况
2. 查看本月TOP5重点工作
3. 进入具体项目工作台

**项目工作**:
1. 打开 `02-客户项目/P001-东义项目/00-项目工作台.md`

## 🎯 使用场景

```
你: "更新东义项目M1进度"
Claude Code:
1. 读取项目工作台
2. 更新进度字段

```

### 场景2: 商务准备
```
你: "准备明天与东义的商务谈判"
Claude Code:
1. 读取商务阶段文档
2. 查阅商务策略库
3. 生成准备清单和要点
4. 提供应对策略建议
```

### 场景3: 会议记录
```
你: "整理今天的项目例会记录" + [原始记录]
Claude Code:
1. 结构化整理记录
2. 更新到对应项目文档
3. 提取待办事项
4. Git提交更新
```

### 场景4: 知识沉淀
```
你: "把这次商务谈判的经验提取到知识库"
Claude Code:
1. 分析谈判过程
2. 提取可复用经验
3. 更新商务策略库
4. 建立关联链接
```
